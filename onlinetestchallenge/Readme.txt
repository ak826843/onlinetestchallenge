Solution to the Crossover's challenge. It's a test application platform developed with Boot Spring, Hibernate, JPA, jQuery, MySQL, Tomcat 7 and Maven.


The file for the definition are in the file:

/CODE/onlinetestchallenge/src/main/resources/application.properties

After configuring the database. You must run the following command at the project root.

mvn package && java -jar target / gas-spring-boot-0.1.0.jar

Now just enter this command on the .jar target folder

java-0.0.1-onlinetestchallenge SNAPSHOT.jar -jar
